To import your new BOARD file into PADS:

1. Unzip the downloaded folder files to a local directory and
Start the PowerPCB program.
2. Select File/Import from the menu at the top of the screen.
3. From the windows screen select the file and path listed above.
The file should have the extension ".asc".
4. You should now have an image of your board available in PADs.

For additional information, please visit this site:
http://www.accelerated-designs.com/help/PowerPCB_BrdSch_import.html

For more information and instructions on creating a library, please
visit this site:
http://www.accelerated-designs.com/help/PowerPCB_import.html


To import your new LIBRARY PARTS to PADS:
1. Start the PowerPCB program.
2. Select File/Library from the menu at the top of the screen.
3. Select the library to translate into from the "Library" list box.
4. Select the Decal icon.
5. Select import from the bottom of the library screen.
6. Select the *.d file included in the zip file.
7. To import a matching symbol with your footprint, simply
repeat the above proceedure (#4 - #6) for the CAE (*.c) and Parts
(*.p) files using the appropriate Icons.

For additional information, please visit this site:
http://www.accelerated-designs.com/help/PowerPCB_import.html

To view a video tutorial, please visit:
http://youtu.be/BT0ao-Y2C-M
